Чтобы поделиться своим локальным хостом
1. Запустите ngrok командою:
ngrok http https://localhost:5001 -subdomain=classesbot
ngrok http https://localhost:5001 -subdomain=classesbot -host-header="localhost:5001"

3. Перейти по адресі:
api.telegram.org/bot<Token>/setwebhook?url=<Domain>
В этом URL-адресе указан ваш бот-токен и ваш адрес пересылки ngrok, использующий https.
(
	https://api.telegram.org/bot<your_token>/setWebHook?url=https://<your_ngrok_url.ngrok.io/
	https://api.telegram.org/bot1054628812:AAHk-NmDCw2JzbYuygug6sW1EIvxr7UCqyI/setwebhook?url=https://classesbot.ngrok.io/api/v1/message/update
)
(
	https://api.telegram.org/bot1054628812:AAHk-NmDCw2JzbYuygug6sW1EIvxr7UCqyI/deleteWebhook
)
